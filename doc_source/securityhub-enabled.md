# securityhub\-enabled<a name="securityhub-enabled"></a>

Checks that AWS Security Hub is enabled for an AWS Account\. The rule is NON\_COMPLIANT if AWS Security Hub is not enabled\. 

**Identifier:** SECURITYHUB\_ENABLED

**Trigger type:** Periodic

**AWS Region:** All supported AWS regions except China \(Beijing\), Asia Pacific \(Jakarta\), Africa \(Cape Town\), Middle East \(UAE\), Asia Pacific \(Osaka\), Europe \(Milan\), Europe \(Spain\), China \(Ningxia\), Europe \(Zurich\) Region

**Parameters:**

None
